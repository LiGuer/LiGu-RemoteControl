# LiGu-RemoteControl  
## Introduction  
* This is a Remote Control Program
* Function:  
  1. [dir "url"]指令: 查询文件列表  
    ·1-1. "url": 目标文件夹路径  
    ·1-2. 未填"url"时,默认"D:\\"  
  2. [file "url"]指令: 文件传输  
    ·2-1. "url": 目标文件路径  
    ·2-2. "url"错误时，会报错，不影响程序  
  3. [quit]指令: 断开连接  
    ·3-1.程序不易关闭，断开连接不影响程序，可多次重连。  
  4. [request]指令: 询问  
  5. [screen]指令: 屏幕截图  
  6. 控件台窗口最小化，避免视觉干扰。  
  7. 套接端口号为57953，位数较高。
